# サプライズ忍者（秘伝・乱入之巻）
Surprise Ninja: Scroll of Intrusion

## 概要
日常の「いまの行動」に対して、ガチムチ忍者がはちゃめちゃに乱入する“乱入シーン＆プロセス”をガチャで生成し、
「この瞬間、忍者が来た方が面白くなる？」を YES/NO で記録するジョークアプリです。

## 機能
- **乱入ガチャ**: いまの行動を入力すると、忍者が乱入してきます。
- **秘伝書UI**: 古びた巻物風のこだわりのデザイン。
- **記録**: 乱入の結果をローカルストレージに保存し、巻物庫（履歴）で閲覧可能。

## 技術スタック
- HTML5 / CSS3 / Vanilla JS
- LocalStorage

## インストール・実行
`index.html` をブラウザで開くだけで動作します。
